Layout
--------------------------
<div>:

    Purpose: A generic container element that is used to group and style sections of content. It is a block-level element.
Attributes:
    id: Unique identifier for the element, useful for CSS styling and JavaScript interactions (e.g., id="content").
    class: Specifies one or more class names for styling (e.g., class="container").
===========================================================================================================================================================================

<span>:

    Purpose: An inline container used to wrap small pieces of content for styling purposes without breaking the line. It is an inline element.
Attributes:
    id: Unique identifier for the element.
    class: Specifies one or more class names for styling.
===========================================================================================================================================================================

Semantic Elements:

    <header>: Represents introductory content or a set of navigational links. Typically contains the site's logo, navigation, and other introductory information.
    <footer>: Represents the footer of a document or section, often containing information about the author, copyright, or links to related documents.
    <nav>: Represents a section of a page intended for navigation links, either within the current document or to other documents.
    <section>: Represents a thematic grouping of content, typically with a heading. It's a block-level element that groups related content.
    <article>: Represents a self-contained piece of content, such as a blog post, article, or news item. Each <article> should make sense on its own, independent of the rest of the document.

Common Attributes for Semantic Elements:
    id: Unique identifier for the element.
    class: Specifies one or more class names for styling.
===========================================================================================================================================================================
