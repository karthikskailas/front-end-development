Links & Media
--------------
<a>:
    Purpose: Defines a hyperlink, which links to another document or location.
Attributes:
    href: Specifies the URL of the linked document (e.g., href="https://example.com").
    target: Specifies where to open the linked document (e.g., target="_blank" opens the link in a new tab).
    rel: Specifies the relationship between the current document and the linked document (e.g., rel="noopener noreferrer" for security and performance reasons when using target="_blank").
    title: Provides additional information about the link, typically displayed as a tooltip on hover.

===========================================================================================================================================================================

<img>:
    Purpose: Embeds an image in the document.
Attributes:
    src: Specifies the path to the image file (e.g., src="image.jpg").
    alt: Provides alternative text for the image if it cannot be displayed (important for accessibility) (e.g., alt="Description of the image").
    width and height: Specifies the width and height of the image (e.g., width="600" or width="50%").
    title: Provides additional information about the image, displayed as a tooltip.

===========================================================================================================================================================================

<video>:
    Purpose: Embeds a video file.
Attributes:
    src: Specifies the path to the video file (e.g., src="video.mp4").
    controls: Adds video controls like play, pause, volume, etc.
    autoplay: Starts playing the video as soon as it is ready.
    loop: Repeats the video indefinitely.
    muted: Mutes the audio of the video by default.
    poster: Specifies an image to show while the video is downloading or until the user clicks play.

===========================================================================================================================================================================

<audio>:
    Purpose: Embeds an audio file.
Attributes:
    src: Specifies the path to the audio file (e.g., src="audio.mp3").
    controls: Adds audio controls like play, pause, volume, etc.
    autoplay: Starts playing the audio as soon as it is ready.
    loop: Repeats the audio indefinitely.
    muted: Mutes the audio by default.
===========================================================================================================================================================================
