Text & Typography
---------------------

<h1> to <h6>:

    Purpose: Define headings, with <h1> being the highest (most important) and <h6> the lowest. They help structure content hierarchically.
Attributes:
    id: Unique identifier for the element, useful for styling and scripting (e.g., id="main-heading").
    class: Specifies one or more class names for CSS styling (e.g., class="headline").

===========================================================================================================================================================================

<p>:

    Purpose: Represents a paragraph of text. It automatically adds space above and below the content.
Attributes:
    id: Unique identifier for the element.
    class: Specifies one or more class names for styling.

===========================================================================================================================================================================

<strong>:

    Purpose: Defines text with strong importance, usually displayed in bold.
    Attributes: None.

===========================================================================================================================================================================

<em>:

    Purpose: Defines emphasized text, usually displayed in italics.
    Attributes: None.

===========================================================================================================================================================================

<br>:

    Purpose: Inserts a line break. Itâ€™s a self-closing tag and does not contain any content.
    Attributes: None.

===========================================================================================================================================================================

<blockquote>:

    Purpose: Represents a section quoted from another source. Often displayed with indentation.
Attributes:
    cite: URL of the source of the quotation (e.g., cite="https://example.com").

===========================================================================================================================================================================

<ul>, <ol>, <li>:

Purpose:
    <ul>: Creates an unordered (bulleted) list.
    <ol>: Creates an ordered (numbered) list.
    <li>: Defines a list item within either <ul> or <ol>.

Attributes for <ol>:
    type: Specifies the type of numbering (e.g., type="1" for numbers, type="A" for uppercase letters).
    start: Specifies the starting value of the list (e.g., start="3" begins numbering at 3).
